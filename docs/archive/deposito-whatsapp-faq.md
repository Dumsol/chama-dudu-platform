# FAQ de Deposito no WhatsApp

Este FAQ e usado pelo bot no fluxo de deposito quando o usuario envia `ajuda`.
O envio acontece em blocos curtos para manter leitura facil no WhatsApp.

## Como abrir, fechar e consultar status
- `abrir`
- `fechar`
- `status`

## Como operar pedido
- `pedidos` ou `pedido atual`: mostra o pedido em andamento.
- `aceitar`: confirma pedido notificado.
- `recusar`: inicia recusa e pede motivo curto.
- `separando`: marca inicio de preparo.
- `eta 20`: define previsao em minutos.
- `saiu`: marca saida para entrega.
- `entregue`: conclui pedido.

## Pausa temporaria
- `pausar`
- `pausa 30`
- `pausa 60`
- `pausa 120`

Use pausa para segurar novos pedidos sem fechar totalmente o deposito.

## Troubleshooting rapido
- Se perder contexto: envie `status` ou `pedidos`.
- Se um passo falhar: repita o comando com texto curto.
- Se precisar revisar instrucoes: envie `ajuda`.

## Boas praticas operacionais
- Atualize ETA real.
- Recuse com motivo claro quando necessario.
- Finalize com `entregue` assim que concluir.
- Evite deixar pedido aberto sem atualizacao.
